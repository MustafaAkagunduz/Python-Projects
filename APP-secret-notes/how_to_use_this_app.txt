ENG
How to use the app:

Encryption:
1-Enter the title.
2-Enter the message to be encrypted.
3-Enter the Key as necessary to decrypt it and don't forget it
4-Press the “Save and Encyrpt” button.

The message is saved in encrypted form in the directory where the file from which the code is run is located.

Decoding:

1-Copy the encrypted text from the resulting .txt file
2- Paste it in the "Enter Your Secret" box.
3-Enter the key you have previously determined.
4-Press the “Decyrpt” button.
